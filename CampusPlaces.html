About

Welcome to Campus Places! Campus Places is a web app that provides several ways to interact with a list of the points of interest on the Wellesley campus. Below, you will find an explanation of how you can interact with the list.

Map View

Access the list via Google Maps and plan a trip! Each location has a marker. When clicked, each marker will display an info window complete with an image and description of the location.

Slideshow

Are you more of a visual person? Ignore text and shift a swipeable (left and right) slideshow. If a spot catches your eyes, simply tap on the image for a popup description.

List with Thumbnails

Do want to to access a straightforward list with pictures without all the fussy details? This option does exactly this and divides points of interest by their location (east or west cmapus).

Pick from Image Gallery

Want more pictures but on one screen? This option has two times more images than Slideshow and provides a caption of the location's description.

Surprise Me!

Can't decide? Let the app choose for you!
